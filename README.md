## Project Overview

This project presents an **interactive SuperStore Sales Dashboard** built using **Microsoft Power BI** to analyze retail business performance.

The dashboard transforms raw sales data into meaningful visual insights that help understand **sales trends, profitability, and regional performance**, enabling better business decision-making.

It demonstrates an end-to-end **Business Intelligence workflow**, including:

- Data Cleaning
- Data Transformation
- KPI Creation
- Interactive Visualization Design

---

## Project Objectives

- Monitor overall sales and profit performance
- Analyze category-wise and region-wise sales trends
- Identify top-performing and loss-making products
- Track yearly performance growth
- Enable interactive data exploration using filters and slicers

---

- **Microsoft Excel**
- **Data Visualization Techniques**
